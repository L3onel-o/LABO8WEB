¿Cuál es la diferencia entre autenticación y autorizacion?

La autenticación es el proceso de verificar la identidad de un usuario, es decir, confirmar que la persona es quien dice ser.

Autorización es el proceso que determina qué recursos o acciones puede realizar un usuario una vez autenticado. Es decir, verifica si el usuario tiene permisos para acceder a ciertas rutas o funcionalidades.

Digamos que => Autenticacion: Quien eres? 
Login: Que puedes hacer?


¿Cuál es la función del token JWT en la guía?

El token JWT (JSON Web Token) tiene la función de autenticar y autorizar las peticiones del usuario de forma segura y sin necesidad de mantener sesiones en el servidor
